Explicação do Projeto



Objetivo do Projeto

Este projeto foi desenvolvido como parte da N1 da matéria de Desenvolvimento Web, ministrada pelo professor Eduardo. A disciplina, de caráter optativo, oferece aos alunos a oportunidade de consolidar e aplicar na prática os conceitos aprendidos ao longo do curso. A razão por trás da escolha deste projeto é justamente o desejo de colocar em prática todas as habilidades adquiridas durante as aulas, além de oferecer uma experiência interativa e divertida para o usuário. O projeto explora as possibilidades da fusão entre criatividade, programação e as tecnologias da web. A motivação para o desenvolvimento desta aplicação surgiu da necessidade de dominar a integração de HTML, CSS e JavaScript, ferramentas fundamentais para o desenvolvimento web. Ao longo do curso, aprendemos a construir interfaces de usuário, manipular dados dinamicamente e criar interações intuitivas para melhorar a experiência do usuário. Este projeto se propõe a unir esses conhecimentos em um formato leve e acessível, mas que, ao mesmo tempo, desafia o aluno a pensar em como criar uma aplicação funcional e divertida. A aplicação consiste em uma brincadeira interativa, onde o usuário será apresentado a animais gerados pela fusão de duas espécies distintas, utilizando inteligência artificial (IA). Por exemplo, uma das combinações pode ser um cavalo com o rabo de um peixe. Após visualizar o animal na tela, o usuário tem a tarefa de dar um nome criativo para a nova criatura gerada. O objetivo principal é a diversão e a criatividade dos participantes, incentivando o engajamento com a aplicação. Ao final do processo, todos os animais criados e seus respectivos nomes dados pelo usuário são exibidos em uma tabela, proporcionando uma visão geral dos resultados e permitindo a revisão das criações de maneira organizada. Este projeto foi inteiramente desenvolvido utilizando as linguagens HTML, CSS e JavaScript, combinando a simplicidade estrutural com a versatilidade na criação de interfaces e interações dinâmicas. Em resumo, este projeto não só coloca em prática os conhecimentos adquiridos durante o curso, mas também busca incentivar a criatividade do usuário através de uma abordagem lúdica e interativa, permitindo o domínio técnico e o desenvolvimento de uma aplicação completa, do início ao fim.
